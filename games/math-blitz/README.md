# ⚡ Math Blitz

## Overview
Math Blitz is a lightning-fast math game that displays one question at a time with full-screen focus. Players build streaks by answering consecutively correct questions and receive bonus feedback for fast answers under 3 seconds.

## Learning Objectives

### Primary Skills
- **Rapid Recall**: Instant recognition of multiplication facts
- **Mental Agility**: Quick calculation of addition and subtraction
- **Focus & Concentration**: Single-task attention without distraction

### Secondary Skills
- **Sustained Attention**: Maintain focus across 20 questions
- **Performance Under Pressure**: Answer accurately with time constraints
- **Self-Monitoring**: Track streaks and personal performance

## Game Mechanics

### Core Features
- **One-at-a-Time Display**: Full attention on current question
- **Streak Counter**: Visual feedback for consecutive correct answers
- **Speed Bonuses**: "FAST!" indicator for answers under 3 seconds
- **Animated Feedback**: Immediate visual confirmation of correctness
- **Progressive Difficulty**: Same 3-level system as other games

### Difficulty Levels

#### Easy
- Multiplication: Tables 2-6
- Addition/Subtraction: 2-digit numbers
- Perfect for building confidence

#### Normal
- Multiplication: Tables 2-10
- Addition/Subtraction: 2-3 digit numbers
- Balanced challenge

#### Hard
- Multiplication: All tables 2-12
- Addition/Subtraction: 3-digit numbers
- Emphasis on difficult tables (7, 9, 11, 12)

### Game Rules
1. **20 Questions**: 80% multiplication, 20% addition/subtraction
2. **2-Minute Limit**: Complete all questions before time runs out
3. **Sequential**: Questions appear one at a time
4. **Streak Tracking**: Count resets on wrong answers
5. **Leaderboard**: 100% accuracy required

## How to Play

1. **Welcome**: Review instructions on opening screen
2. **First Question**: Timer starts, question appears
3. **Select Answer**: Click/tap the correct answer
4. **Instant Feedback**: See ✓ or ✗ with animation
5. **Build Streaks**: String together correct answers
6. **Speed Bonuses**: Answer in <3 seconds for "FAST!" message
7. **Continue**: Auto-advance to next question
8. **Complete**: Finish all 20 questions or run out of time
9. **Results**: View stats including max streak and fast answers
10. **Leaderboard**: Check ranking if you scored 100%

## Pedagogical Framework

### Evidence-Based Design

**Retrieval Practice**
- Active recall strengthens memory
- Testing effect improves long-term retention
- Spaced repetition through repeated play

**Immediate Feedback**
- Corrects errors before they solidify
- Reinforces correct responses
- Reduces anxiety through certainty

**Desirable Difficulties**
- Time pressure creates productive challenge
- Varied operations prevent mechanical responding
- Interleaved practice improves discrimination

**Flow State**
- Clear goals (answer correctly)
- Immediate feedback (✓/✗)
- Challenge matches skill level (3 difficulties)
- Total focus (one question at a time)

### Learning Progression

**Stage 1: Accuracy Focus**
- Goal: Complete all questions correctly
- Don't worry about time or speed
- Build confidence with easy mode

**Stage 2: Fluency Building**
- Goal: Reduce hesitation time
- Notice which facts are automatic
- Practice problematic facts separately

**Stage 3: Speed Development**
- Goal: Increase fast answers (<3 sec)
- Maintain 100% accuracy
- Achieve high streaks

**Stage 4: Mastery**
- Goal: Consistent perfect scores
- Fast completion times
- Automaticity across all facts

## Common Student Questions

**Q: Does a streak matter for the leaderboard?**
A: No, only 100% accuracy and time matter. But high streaks indicate strong performance!

**Q: What if I'm not getting fast answers?**
A: That's okay! Speed develops naturally with practice. Focus on accuracy first.

**Q: Can I go back to a previous question?**
A: No, Math Blitz is forward-only. This encourages decisive answering.

**Q: What happens if time runs out?**
A: The game ends and you see results, but incomplete games aren't added to the leaderboard.
